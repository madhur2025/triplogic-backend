CORS (Cross-Origin Resource Sharing) : CORS ek security rule hai jo decide karta hai kaunsa frontend tumhare backend se data le sakta hai

request flow me cors kaha aate hai
Client → (CORS check) → Server → Routes → Controller → Model → Database

⚠️ cors backend k logic ka part nhi hota, ye ek security layer hai browser side ka.

cors ko use krne k baad ab backend ko koi bhi use kr skta hai, isliye cors me origin set krte hai taki jis frontend ko humne allow kiya hai sirf wohi is backend ko access kr skte hai.

// method 1 : publically sbko allow kr diya ab koi bhi use kr skta hai.
const cors = require('cors')
app.use(cors());

// method 2 : specific frontend k liye allow kiya hai sirf
const cors = require('cors')

app.use(cors({
    origin: "http://localhost:5173"
}))

// method 3 : multiple frontend k liye allow kr diya hai.
app.use(cors({
    origin: ["http://localhost:5173", "https://myapp.com"]
}))
